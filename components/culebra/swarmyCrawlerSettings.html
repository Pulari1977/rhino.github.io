Swarmy Crawler Settings

Component Index > Culebra > Behaviors > Swarmy Crawler Settings

Settings for the Swarmy Crawler Swarm System

Inputs

NameIDDescriptionType
ConnectCInput a boolean toggle - True = Connect the heads (visualizes their search radius) | False = Do not draw connectivityBoolean
SwarmSInput a boolean toggle - True = Swarm Behavior | False = Move without Swarm behaviorBoolean
Connectivity ThresholdCTInput a float value specifying the connectivity threshold (This number should be the same as the search radius)Number
SpeedMVInput a float value specifying the distance each creeper can move per cycleNumber
Search RadiusSRInput a float value specifying the distance each creeper can seeNumber
Align ValueAVInput a float value specifying alignment vector scale valueNumber
Separation ValueSVInput a float value specifying separation vector scale valueNumber
Cohesion ValueCVInput a float value specifying cohesion vector scale valueNumber
MultiplierMInput a float value specifying vector multiplier valueNumber

Outputs

NameIDDescriptionType
Behavioral SettingsBSOutputs settings list values for Creeper EngineGoo
