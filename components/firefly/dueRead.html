Due Read

Component Index > Firefly > Arduino & I/O Boards > Due Read

This component will read values to all corresponding digital
and analog pins on an Arduino Mega board. Use the Timer
component (Parameters/Special/Timer) to update the
sensor values in real-time. Note: This component is meant
to be used in tandem with the Firefly Firmata Arduino
Sketch which can be downloaded from the Firefly website
at: http://www.fireflyexperiments.com.
