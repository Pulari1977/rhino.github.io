EnergyPlus Shade Material

Component Index > Honeybee > 06 | Energy | Material | Construction > EnergyPlus Shade Material

Use this component to create a custom material for shades, which can be plugged into the "Honeybee_EnergyPlus Window Shade Generator" component.
In order to apply the material to a window shade and adjust geometric characteristics of the shade, you should plug the output of this component into a "Honeybee_EnergyPlus Window Shade Generator" component.
Note that the material characteristics here can refer to either blind slats, roller shades, perforated exterior metal screens, or the properties of electrochromic glazing in an "on" state.

Inputs

NameIDDescriptionType
materialName_materialName_An optional name for the shade material.Goo
reflectance_reflectance_A number between 0 and 1 that represents the front reflectance of the shade material. The default value is set to 0.65.Goo
transmittance_transmittance_A number between 0 and 1 that represents the transmittance of the shade material. The default value is set to 0 for a perfectly opaque shade.Goo
emissivity_emissivity_A number between 0 and 1 that represents the emissivity of the shade material. The default value is set to 0.9 for a non-metalic shade.Goo
thickness_thickness_An optional number representing the thickness of the shade in meters. For blinds, this is the thickness of each blind slat and, for roller shades and screens, this is the thickness of the fabric or screen material. For electrochromic windows, this variable is discounted since window materials with n mass are used. The default is set to 0.00025 m for a very thin shade.Goo
conductivity_conductivity_An optional number representing the conductivity of the shade material in W/m-K. The default is set to 221 W/m-K.Goo

Outputs

NameIDDescriptionType
shadeMaterialshadeMaterialA shade material that can be plugged into the ZoneShades component.Goo
