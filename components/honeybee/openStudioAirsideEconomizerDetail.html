Inputs

NameIDDescriptionType
_uniqueName_uniqueNamea required field to uniquely name the economizerGoo
_economizerControlType__economizerControlType_... requires an integer specifying the type of economizer 0:FixedDryBulb(default),1:DifferentialDryBulb,2:FixedEnthalpy,3:DifferentialEnthalpy,4:ElectronicEnthalpy,5:FixedDewPointAndDryBulb,6:DifferentialDryBulbAndEnthalpy,7:NoEconomizerGoo
_controlAction__controlAction_... Requires an integer. See ecdict for different values to supply. Supply nothing and it defaults to "ModulateFlow"Goo
_maximumAirFlowRate__maximumAirFlowRate_... supply nothing and it will Autosize (recommended)Goo
_minimumAirFlowRate__minimumAirFlowRate_... do nothing and it will Autosize (recommended)Goo
_minimumLimitType__minimumLimitType_... do nothing and it defaults to Proportional Minimum (min depends on the supply air flow rate as opposed to an absolute number)Goo
_minimumOutdoorAirSchedule__minimumOutdoorAirSchedule_... This is a schedule with values between 0 and 1, and it is multiplied by the minimumAirFlowRate. It is usually left blank, but can be used to fine tune the economizer during warm-up time or after hours.Goo
_minimumOutdoorAirFracSchedule__minimumOutdoorAirFracSchedule_... this overrides minOutdoorAirSchedule and minAirflowRate. It is a schedule between 0 and 1. It is often used to create a 100% outside air system.Goo
_maximumOutdoorAirFracSchedule__maximumOutdoorAirFracSchedule_... this is a schedule between 0 and 1. It is often used to create a recirculating outside air system such as that in patient rooms.Goo
_maximumLimitDewpoint__maximumLimitDewpoint_... needed for when the ControlType is Fixed Dewpoint and Dry Bulb. Otherwise leave blankGoo
_sensedMinimum__sensedMinimum_... is the minimum of whatever the control type, at this point the system goes to minimum flowGoo
_sensedMaximum__sensedMaximum_... is the maximum of whatever the control type, at this point the system goes to minimum flowGoo
_economizerLockoutMethod__economizerLockoutMethod_... should only used when the HVAC system is packaged DXGoo
_timeOfDaySchedule__timeOfDaySchedule_this field is only used when the outdoor flow rate is based on a schedule. It is rare for a normal economizer to have this value set. If so, apply the name of a schedule.Goo
_mechVentController__mechVentController_an optional field, though highly recommended. Open Studio provides default behavoir for this controller.Goo
_availabilityManagerList__availabilityManagerList_allows you to toggle between different AvailabilityManagers. Right now, we simply allow you to create a list that has only one AvailabilityManager, and the type of manager can be ScheduledOrNightCycleGoo
