EPlus Hot Water Boiler

Inputs

NameIDDescriptionType
_name_name... provide a unique name for each boiler that you specifyGoo
_sequence_sequence...is a placeholder now (defaulted always to zero. should allow users to create multiple boilers and assign sequencing capabilities. Must be an integer, including zero.Goo
_fuelType__fuelType_... Leave blank and the default is NaturalGas. Choices are 0=Electricity, 1=NaturalGas, 2=PropaneGas, 3=FuelOil#1, 4=FuelOil#2, 5=Coal, 6=Diesel, 7=Gasoline, 8=OtherFuel1, 9=OtherFuel2Goo
_nominalCapacity__nominalCapacity_...If left blank, a boiler will be autosized. However, entering a value allows the capacity to be user-defined, (the Units are in Watts).Goo
_sizingFactor__sizingFactor_... a dimensionless number that will be multiplied by the capacity and the design water flow rate. Usually will be something like "1.1" (a 10% increase).Goo
_nominalEfficiency__nominalEfficiency_... The thermal capacity of the boiler. A value between 0 and 1.Goo
_designOutletTemperature__designOutletTemperature_... If left blank?, otherwise enter a value to specify the leaving temp at design, in CelsiusGoo
_designWaterFlowRate__designWaterFlowRate_... If left blank ?, the water flow rate will autosize. Otherwise enter a value to specify the design water flow rate (units are meters cubed per second)Goo
_minPartLoadRatio__minPartLoadRatio_... Specify a value for the boiler turndown.Goo
_maxPartLoadRatio__maxPartLoadRatio_... Specify a value for the max boiler capacity (cannot exceed 1.1)Goo
_optimumPartLoadRatio__optimumPartLoadRatio_... Specify a value for the ideal operating part load ration (between min and max part load ratio)Goo
_outletTempMaximum__outletTempMaximum_... If left blank, the value is 99 degrees Celsius. Otherwise provide your own value for the maximum temperature out of the boiler. This value cannot exceed 99.Goo
_boilerFlowMode__boilerFlowMode_... The default, if not specified, is "NotModulated". However, there are three available choices. "NotModulated", "ConstantFlow", and "LeavingSetpointModulated"Goo
_parasiticElectricLoad__parasiticElectricLoad_... The default, if not specified, is 0 (zero). Provide a value to indicate, in Watts, how much parasitic power is consumed by the boiler by controls, fans, or pumps integral to the boiler.Goo
_curveTemperatureVariable__curveTemperatureVariable_... There are two options: "EnteringBoiler" and "LeavingBoiler". This value is used to control which value of hot water to use when evaluating efficiency curves (if provided). _Curves_ ... Not yet implemented. Allows you to specify custom part load curves for DX coils.Goo
_Curves__Curves_Script variable PythonGoo
