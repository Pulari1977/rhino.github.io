User Iso-Lines (karamba)

Component Index > Karamba > Utilities > User Iso-Lines (karamba)

Returns iso-lines at intersection point of given line with shell based on user supplied values.

Inputs

NameIDDescriptionType
ModelModelModel with shellsModel
ValuesValsValues used for constructing the iso-lines: one for each node of the model.Number
List of sources of iso lines (Lines|Points)SourcesList of lines that intersect the shell or points close to it that define the source of a stream line.Goo
Segment LengthSeg-LApproximate length of the segments of the iso-lines [m].Number
SmoothingSmoothIf 'true' force flow lines output are splines otherwise lists of straight lines.Boolean

Outputs

NameIDDescriptionType
ModelModelModel with shellsModel
LLinesIso-lines: There are two halves per line which meet at the intersection point.Curve
VValueValue at the corresponding iso-line. There are two values per iso-line in order to match the structure of the iso-line output.Number
