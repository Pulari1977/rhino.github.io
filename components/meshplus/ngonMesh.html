ngon Mesh

Component Index > MeshPlus > Make > ngon Mesh

This experimental component creates a topology map for mesh comprised of faces of with a vertex count of 3 or higher. This component takes a list of polylines with coincident control points and merges these points into a mesh, creating unified vertices, edge, faces, and normals as well as a map of the topological relationships between the elements.
